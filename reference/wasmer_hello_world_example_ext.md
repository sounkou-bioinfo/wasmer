# Hello World example

Create a simple WASM "Hello World" example.

## Usage

``` r
wasmer_hello_world_example_ext(ptr)
```

## Arguments

- ptr:

  External pointer to WasmerRuntime.

## Value

String result from WASM hello function

## Details

Create a simple "Hello World" example
