# Instantiate a compiled module in the runtime.

Instantiate a compiled module in the runtime.

## Usage

``` r
wasmer_instantiate_ext(ptr, module_name, instance_name)
```

## Arguments

- ptr:

  External pointer to WasmerRuntime

- module_name:

  Name of the module to instantiate

- instance_name:

  Name to register the instance under

## Value

Status message
